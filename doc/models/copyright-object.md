
# Copyright Object

## Structure

`CopyrightObject`

## Fields

| Name | Type | Tags | Description |
|  --- | --- | --- | --- |
| `text` | `String` | Optional | The copyright text for this content. |
| `type` | `String` | Optional | The type of copyright: `C` = the copyright, `P` = the sound recording (performance) copyright. |

## Example (as JSON)

```json
{
  "text": "text8",
  "type": "type2"
}
```
